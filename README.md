# radioApp
## Simple swift App to stream SVT Radio P2 

This is App is write Swift 5, allow stream SVT Radio P2. Is a simple example how stream sound in Swift.
Include buttons:

* Volume comtrol
* Play-Pause

Streaming using AVPlayer class.
